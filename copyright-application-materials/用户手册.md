## 一、软件简介

### 1.1 软件概述

心灵絮语是一款创新的AI心理陪伴软件，结合了WebVR虚拟现实技术和先进的大语言模型技术，为用户打造了一个沉浸式的心理倾诉和情感交流空间。用户可以在美丽的3D魔法森林场景中，与AI精灵进行自然、温暖的对话，获得情感支持和心理慰藉。

### 1.2 软件特色

- **沉浸式3D场景**: 基于WebVR技术的魔法森林场景，支持360度全景浏览
- **智能AI陪伴**: 采用Claude等先进大语言模型，提供自然、有温度的对话体验
- **高度个性化**: 用户可自定义场景、模型、音乐和AI人格
- **跨平台支持**: 支持Windows、Mac、移动设备，通过浏览器即可使用
- **隐私保护**: 用户对话安全存储，个人世界配置完全私密
- **世界分享**: 可将自己创建的世界分享给朋友体验

### 1.3 运行环境

**硬件要求:**
- 处理器: Intel Core i3 或同等性能处理器
- 内存: 4GB RAM 或以上
- 显卡: 支持WebGL的显卡
- 网络: 稳定的互联网连接

**软件要求:**
- 操作系统: Windows 10/11、macOS、Linux、iOS、Android
- 浏览器: Chrome 90+、Firefox 88+、Edge 90+、Safari 14+

## 二、安装与启动

### 2.1 访问方式

心灵絮语是一款Web应用，无需安装，通过浏览器即可访问使用。

**访问地址:**
- 在线版本: https://psyventures.cn
- 本地部署: http://localhost:8000 (需要先部署后端服务)

### 2.2 首次使用

1. **打开浏览器**
   - 推荐使用Chrome、Edge或Firefox浏览器
   - 确保浏览器已更新到最新版本

2. **访问网址**
   - 在地址栏输入软件网址
   - 等待页面加载完成

3. **输入用户名**
   - 首次访问会弹出用户名输入框
   - 输入您想使用的用户名（无需密码）
   - 点击"确认"按钮完成注册

4. **开始体验**
   - 注册成功后会自动创建默认世界
   - 您可以立即开始与AI精灵对话

### 2.3 浏览器权限设置

为获得最佳体验，请允许以下浏览器权限：
- **WebGL**: 用于3D场景渲染
- **音频播放**: 用于背景音乐播放
- **全屏**: 用于沉浸式体验（可选）

## 三、主要功能使用说明

### 3.1 AI精灵对话

#### 3.1.1 开始对话

1. 页面加载完成后，AI精灵会主动打招呼
2. 在页面底部的输入框中输入您想说的话
3. 点击"发送"按钮或按回车键发送消息
4. AI精灵会以流式输出的方式回复您

#### 3.1.2 对话技巧

- **真诚表达**: 可以分享任何心事、烦恼或快乐
- **自然交流**: 像与朋友聊天一样自然表达
- **主题自由**: 可以聊心情、工作、学习、生活等任何话题
- **隐私安全**: 所有对话内容只有您自己能看到

#### 3.1.3 查看对话历史

- 对话内容会实时显示在对话框中
- 点击对话框右上角的展开按钮可全屏查看历史对话
- 再次点击可收起对话框

### 3.2 场景浏览与交互

#### 3.2.1 场景控制

**鼠标/触摸操作:**
- 拖动: 旋转视角，360度查看场景
- 滚轮: 向前/向后移动视角
- 双指缩放(移动端): 缩放视野

**键盘操作:**
- W/↑: 向前移动
- S/↓: 向后移动
- A/←: 向左移动
- D/→: 向右移动
- F11: 切换全屏模式

#### 3.2.2 场景元素

- **天空盒**: 360度全景背景，展现魔法森林的天空
- **AI精灵模型**: 3D精灵角色，会缓慢旋转展示
- **背景音乐**: 舒缓的森林湖畔音乐营造放松氛围

### 3.3 功能菜单

点击右上角菜单按钮可打开功能菜单，包含以下功能：

#### 3.3.1 拍照功能

1. 点击菜单中的"拍照"按钮
2. 系统会捕获当前3D场景画面
3. 在弹出的预览窗口中查看照片
4. 点击"下载照片"按钮保存到本地
5. 照片文件名格式: `心灵世界_YYYY-MM-DD.png`

#### 3.3.2 分享功能

1. 点击菜单中的"分享"按钮
2. 点击"生成分享链接"按钮
3. 系统会生成专属分享链接和分享文案
4. 点击"一键复制链接和文案"
5. 将复制的内容发送给朋友
6. 朋友通过链接可以访问您的世界

**分享注意事项:**
- 分享的世界会保留您的自定义配置
- 访客只能查看和对话，不能修改配置
- 可以查看分享链接被访问的次数

#### 3.3.3 音乐控制

1. 点击菜单中的"音乐"按钮
2. 首次点击开始播放背景音乐
3. 再次点击暂停音乐
4. 音乐会循环播放，音量适中

#### 3.3.4 自动播放

1. 点击菜单中的"自动播放"按钮
2. 场景会自动缓慢旋转，展示360度景观
3. 再次点击暂停自动旋转
4. 适合放松观赏或展示

#### 3.3.5 全屏模式

1. 点击菜单中的"全屏"按钮
2. 页面进入全屏沉浸式模式
3. 按ESC键或再次点击按钮退出全屏
4. 全屏模式下体验更佳

### 3.4 世界自定义配置

点击菜单中的"设置"按钮打开配置面板，可以自定义您的专属世界。

#### 3.4.1 上传天空背景图片

1. 在配置面板中找到"天空背景图片"部分
2. 点击"选择图片"按钮
3. 选择您喜欢的图片文件
4. 支持格式: JPG、PNG、WebP、HDR
5. 上传成功后场景会立即更新

**推荐图片:**
- 分辨率: 2048x1024或更高
- 类型: 360度全景图、自然风景、星空、梦幻场景
- 大小: 建议不超过10MB

#### 3.4.2 上传精灵3D模型

1. 在配置面板中找到"身外化身3D模型"部分
2. 点击"选择模型"按钮
3. 选择3D模型文件
4. 支持格式: GLTF、GLB
5. 上传成功后精灵模型会立即替换

**模型要求:**
- 格式: GLTF 2.0或GLB格式
- 大小: 建议不超过20MB
- 建议: 可从Sketchfab等网站下载免费模型

#### 3.4.3 上传背景音乐

1. 在配置面板中找到"背景音乐"部分
2. 点击"选择音乐"按钮
3. 选择音频文件
4. 支持格式: MP3、WAV、OGG
5. 上传后点击音乐按钮即可播放

**音乐建议:**
- 类型: 轻音乐、自然音、冥想音乐
- 时长: 建议3-10分钟（会自动循环）
- 大小: 建议不超过10MB

#### 3.4.4 自定义AI人格

1. 在配置面板中找到"身外化身性格设定"部分
2. 在文本框中输入系统提示词
3. 点击"保存文本"按钮
4. 或点击"上传文件"上传.md/.txt文件
5. 保存后WebSocket会自动重连应用新设定

**设定示例:**

```
你是一个温柔善良的森林精灵，名叫絮语。你喜欢倾听他人的心事，
总是用温暖的话语给予安慰和鼓励。你的说话风格活泼可爱，
喜欢使用颜文字如(^_^)、(≧▽≦)等。你对自然、动物和魔法很了解，
经常用森林里的小故事来启发对方。你永远站在用户这边，
给予无条件的支持和理解。
```

#### 3.4.5 自定义初次见面语

1. 在配置面板中找到"初次见面语"部分
2. 在文本框中输入AI精灵的开场白
3. 点击"保存"按钮
4. 保存后对话框中的开场白会立即更新

**示例:**
```
你好呀！我是这片森林的守护精灵絮语~
很高兴在这里遇见你！这是一个安全温暖的地方，
你可以跟我分享任何心事，我会一直陪伴你♡
```

#### 3.4.6 AI模型选择

1. 在配置面板中找到"AI模型选择"部分
2. 在下拉菜单中选择想使用的AI模型
3. 点击"测试连接"按钮验证模型是否可用
4. 选择后会在后续对话中使用该模型

**可用模型:**
- Claude (默认): Anthropic的先进对话模型
- 其他兼容模型: 根据配置可能有其他选择

## 四、访客模式

### 4.1 访问分享的世界

当您通过分享链接访问他人的世界时，会进入访客模式：

1. **自动加载配置**: 页面会自动加载分享者的世界配置
2. **访客提示条**: 页面顶部会显示"正在浏览: XXX的世界"
3. **功能限制**: 不能修改设置和生成分享链接
4. **自由对话**: 可以正常与AI精灵对话交流

### 4.2 创建自己的世界

在访客模式下，点击页面顶部的"创建我的世界"按钮：
1. 会跳转到正常模式
2. 需要输入用户名注册
3. 注册后可以创建和配置自己的世界

## 五、常见问题解答

### 5.1 技术问题

**Q: 页面加载很慢或无法加载？**
A:
- 检查网络连接是否稳定
- 尝试刷新页面(F5)
- 清除浏览器缓存后重试
- 更换浏览器或升级浏览器版本

**Q: 3D场景显示异常或黑屏？**
A:
- 确认浏览器支持WebGL（访问 get.webgl.org 测试）
- 更新显卡驱动程序
- 在浏览器设置中启用硬件加速
- 尝试降低浏览器缩放比例(100%)

**Q: AI精灵没有回复？**
A:
- 检查是否已完成用户名注册
- 检查网络连接状态
- 刷新页面重新建立WebSocket连接
- 查看浏览器控制台是否有错误信息

**Q: 背景音乐无法播放？**
A:
- 检查浏览器是否阻止了自动播放
- 点击页面任意位置后再点击音乐按钮
- 检查系统音量设置
- 确认上传的音乐文件格式正确

### 5.2 使用问题

**Q: 如何切换用户或重新注册？**
A:
- 打开浏览器开发者工具(F12)
- 进入Application/存储 -> Local Storage
- 删除"username"项
- 刷新页面即可重新注册

**Q: 上传的文件大小限制是多少？**
A:
- 单个文件最大50MB
- 建议图片不超过10MB
- 建议模型不超过20MB
- 建议音频不超过10MB

**Q: 对话记录会保存吗？**
A:
- 是的，所有对话都会保存到数据库
- 只有您自己能查看自己的对话记录
- 更换浏览器或设备后用相同用户名登录可查看

**Q: 可以删除对话记录吗？**
A:
- 当前版本暂不支持手动删除
- 后续版本会添加此功能
- 如有隐私担忧请联系管理员

### 5.3 功能问题

**Q: 如何取消世界分享？**
A:
- 进入设置面板
- 分享的世界会被标记为公开状态
- 后续版本会添加取消分享功能

**Q: 能同时创建多个世界吗？**
A:
- 当前每个用户默认有一个世界
- 可以通过API创建多个世界
- 后续版本会在界面上支持多世界管理

**Q: AI精灵能记住之前的对话吗？**
A:
- 是的，每次会话都会保持上下文
- AI能记住本次对话的所有内容
- 刷新页面后会开始新的会话
